Password Strength Validator (C++)

A simple C++ program that checks whether a password is strong based on basic security rules.

Features
Checks for uppercase letters (A–Z)
Checks for lowercase letters (a–z)
Checks for digits (0–9)
Checks for special characters (@, #)
How it works

The program scans each character of the input and verifies if all required categories are present. It prints:

GOOD PASSWORD if all conditions are met
INVALID PASSWORD otherwise
